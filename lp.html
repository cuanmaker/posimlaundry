🔒 Transaksi Aman & Instan Download

Tanya Jawab Seputar Produk

Apakah saya perlu sewa Hosting & Domain bulanan? ▼
TIDAK PERLU! Aplikasi ini dibangun menggunakan infrastruktur Google (Google Sheets sebagai Database dan Google Apps Script sebagai API Server). Semuanya gratis dan menempel pada akun Google Gmail Anda/klien Anda. Anda cukup deploy HTML-nya di Vercel/Netlify (Gratis) atau simpan di lokal komputer kasir.
Saya gak jago ngoding, apakah cara setup-nya susah? ▼
Sangat gampang! Cukup buat Spreadsheet baru, buka menu Extensions > Apps Script, lalu Paste-kan kode Backend yang kami berikan. Setelah itu Deploy dan dapatkan link API-nya. Masukkan link API tersebut ke file HTML-nya. Selesai!
Apa maksudnya dilarang Re-Brand? ▼
Anda bebas mengganti Nama Toko, Alamat, Logo Klien, dan warna Landing Page dari menu Pengaturan di dalam aplikasi (Itu hak klien Anda). Yang dilarang adalah Anda membongkar kode sumbernya lalu menghapus jejak tulisan "Powered by iMersLonry" atau mengklaim bahwa aplikasi inti tersebut adalah buatan Anda sendiri dari nol.
Apakah aplikasi ini berat jika dijalankan di HP Kasir? ▼
Sangat ringan! Aplikasi ini dibangun dengan teknologi modern yang tidak membebani perangkat. Asalkan ada koneksi internet, kasir bisa menginput transaksi dan mencetak struk thermal dengan mulus langsung dari browser HP, Tablet, atau Laptop.
